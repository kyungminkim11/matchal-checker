FAQ · 개인정보

안심하고 사용할 수 있도록 안내합니다

Instagram 비밀번호를 입력해야 하나요?

아니요. 로그인 정보, 인증 코드, 쿠키를 요구하지 않습니다.

ZIP 파일이 서버로 전송되나요?

아니요. ZIP과 작업 기록은 현재 브라우저 안에서만 처리됩니다.

자동으로 언팔해주나요?

아니요. 프로필만 열어주며 실제 작업은 사용자가 직접 진행합니다.

결과가 실시간인가요?

아니요. Instagram 데이터가 생성된 시점을 기준으로 합니다.

다른 계정 기록이 섞이나요?

ZIP 데이터별 작업공간을 만들어 기록을 분리합니다.
